florentdesjardins.fr
====================

My website.

Prerequisites
-------------

- ndenv (https://github.com/riywo/ndenv)

Before starting work
--------------------

```shell
ndenv install
npm install
```

Start the app
-------------

```shell
npm start
```

Build the app during development
--------------------------------

```
npm run build --env dev
```

Clean
-----

```
npm run clean
```

Deploy
------

```
sudo chmod a+x deploy.sh
./deploy.sh
```
